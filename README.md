# 🛒 E-Commerce FAQ Chatbot

A minimalist, AI-powered FAQ chatbot built with Streamlit and Scikit-Learn that provides instant answers to customer queries using cosine similarity.

## 🚀 Features
* **NLP-Powered:** Uses TF-IDF and Cosine Similarity for intelligent response matching.
* **Streamlit UI:** Clean, custom-styled interface for a seamless chat experience.
* **Session Memory:** Retains chat history for better user flow.

## 🛠 Tech Stack
* Python | Streamlit | Scikit-Learn | Pandas
